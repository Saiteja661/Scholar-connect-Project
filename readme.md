-- Steps to run the project locally:
	
	- Extract the zip file
	- Open terminal
	- Navigate to the extracted folder using the command cd <folder path>
	- Once inside the folder in the terminal run the following command:
		- npm install
		- npm run dev
	- On running the "npm run dev" command, you will receive a localhost link in the terminal,
		paste the link in your browser to see the running project
